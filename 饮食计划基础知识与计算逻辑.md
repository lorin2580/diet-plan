# 饮食计划 — 基础知识、计算逻辑与科学依据

> 本文档整理自最新权威研究（2023-2025），为饮食计划网页提供科学依据。

---

## 一、基础代谢率（BMR）计算

### 1.1 公式选择

当前学界共识：**Mifflin-St Jeor 方程（1990）** 是普通成年人最准确的 BMR 预测公式，多项 2024-2025 年研究验证其优于传统的 Harris-Benedict 方程。

| 公式名称 | 年代 | 准确性评价 |
|----------|------|-----------|
| **Mifflin-St Jeor** | 1990 | 普通成年人最准确，60.1%准确率（2025年研究） |
| Harris-Benedict | 1919 | 在老年人(58-78岁)中相关性更好，对亚洲人群普遍高估 |
| Cunningham | 1980 | 运动员群体最优，需要体脂率数据 |

**结论**：默认使用 Mifflin-St Jeor，同时提供 Harris-Benedict 作为备选。

### 1.2 Mifflin-St Jeor 公式

```
男性: BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄(岁) + 5
女性: BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄(岁) - 161
```

### 1.3 必需输入参数

| 参数 | 原因 |
|------|------|
| **性别** | 男女公式不同（肌肉量/激素差异） |
| **体重(kg)** | 代谢最核心驱动因素 |
| **身高(cm)** | 影响体表面积和瘦体重 |
| **年龄(岁)** | 代谢随年龄递减（约每10年降2-5%） |
| **体脂率(%)** | 可选但推荐。瘦体重代谢活跃度是脂肪的3倍，同体重不同体脂率BMR可差200-400kcal |

## 二、每日总能量消耗（TDEE）

### 2.1 公式

```
TDEE = BMR × 活动系数(PAL)
```

### 2.2 活动系数（PAL）标准值

| 活动水平 | PAL系数 | 描述 |
|----------|---------|------|
| 久坐不动 | 1.20 | 极少运动，办公室工作，几乎不活动 |
| 轻度活动 | 1.375 | 每周1-3天轻度运动（散步、瑜伽） |
| 中度活动 | 1.55 | 每周3-5天中等运动（慢跑、力量训练） |
| 高度活跃 | 1.725 | 每周6-7天高强度运动 |
| 极度活跃 | 1.90 | 每天高强度训练+体力劳动（运动员） |

> **注意**：大多数人会高估自己的活动水平，建议保守选择。

## 三、减重热量缺口计算

### 3.1 核心换算

```
1 kg 纯脂肪 ≈ 7700 kcal
减重1kg（含水分）≈ 需创造7700 kcal的热量缺口
```

### 3.2 推荐减重速度

| 每日热量缺口 | 月减重量 | 评价 |
|-------------|---------|------|
| 300 kcal/天 | ~1.2 kg/月 | 温和可持续 |
| 500 kcal/天 | ~2.0 kg/月 | **推荐上限** |
| 750 kcal/天 | ~3.0 kg/月 | 偏激进 |
| 1000 kcal/天 | ~4.0 kg/月 | 需医学监督 |

**安全底线**：
- 女性每日摄入不低于 **1200 kcal**
- 男性每日摄入不低于 **1500 kcal**

### 3.3 代谢适应修正（2024年《柳叶刀》模型）

身体会对热量限制产生代谢适应，减重曲线更接近**对数曲线**：
- 每天500kcal赤字，理论1年减23.7kg，实际约减10.4kg
- 随减重进展，每月需重新计算BMR（体重下降→BMR下降）
- 每减重5kg建议重新评估摄入目标

### 3.4 体脂率与减脂速度上限

```
每kg体脂肪每天最多动员 ≈ 65 kcal
减掉1kg纯脂肪最快天数 = 7700 / (体脂率 × 体重 × 65)
```

例如：60kg、体脂率25% → 7700 / (25%×60×65) ≈ 7.9天/公斤（理论最快）

## 四、三大宏量营养素

### 4.1 AMDR（宏量营养素可接受范围）

依据《中国居民膳食指南（2022）》和《中国居民膳食营养素参考摄入量 DRIs 2023版》：

| 宏量营养素 | 供能占比(AMDR) | 每克热量 |
|-----------|---------------|---------|
| **碳水化合物** | 50%–65% | 4 kcal/g |
| **蛋白质** | 10%–15% | 4 kcal/g |
| **脂肪** | 20%–30% | 9 kcal/g |

### 4.2 蛋白质摄入量（DRIs 2023 更新）

| 人群 | 推荐量 |
|------|--------|
| 普通成人 | 0.8 g/kg/天 |
| 65岁以上老年人 | 1.2–1.5 g/kg/天（2023版上调） |
| 减脂期间 | **1.5–1.8 g/kg/天**（防肌肉流失） |
| 增肌/运动员 | 1.6–2.2 g/kg/天 |

### 4.3 减脂期营养素分配逻辑

```
每日可摄入热量 = TDEE - 热量缺口

步骤1: 先确定蛋白质
  蛋白质(g) = 体重(kg) × 1.5~1.8  (取1.6为默认)
  蛋白质热量 = 蛋白质(g) × 4

步骤2: 再确定脂肪
  脂肪占剩余热量 20%-30%
  脂肪(g) = (可摄入热量 × 0.25) / 9

步骤3: 剩余热量分配给碳水
  碳水热量 = 可摄入热量 - 蛋白质热量 - 脂肪热量
  碳水(g) = 碳水热量 / 4
```

## 五、三种减脂饮食模式

### 5.1 低碳减脂

**适合**：运动频率低（周≤2次）、偏好多肉少主食、胰岛素敏感度低者

| 营养素 | 摄入量 | 热量占比 |
|--------|--------|---------|
| 碳水 | 1.5~2.0 g/kg | 15%~26% |
| 蛋白质 | 1.5~1.8 g/kg | 35%~45% |
| 脂肪 | 剩余热量 | 25%~35% |

> 碳水不能长期 <1.5g/kg，否则代谢下降、皮质醇升高。

### 5.2 高碳减脂

**适合**：高频力量训练（周≥5次）、体力劳动者、偏好主食者

| 营养素 | 摄入量 | 热量占比 |
|--------|--------|---------|
| 碳水 | 2.5~3.5 g/kg | 45%~65% |
| 蛋白质 | 1.1~1.5 g/kg | 20%~30% |
| 脂肪 | 剩余热量 | 15%~20% |

### 5.3 碳循环

**适合**：中等训练频率（周2-4次）、平台期突破、应酬多饮食不可控者

| 日类型 | 碳水(g/kg) | 蛋白质(g/kg) | 脂肪 |
|--------|-----------|-------------|------|
| 低碳日 | 1.0~2.0 | 1.5~2.0 | 适量增加(30-35%) |
| 中碳日 | 2.0~2.5 | 1.5 | 适量(25-30%) |
| 高碳日 | 2.5~3.5 | 1.1~1.5 | 低(15-20%) |

常用周期：低-低-中-高 循环，或训练日高碳+休息日低碳。

> 蛋白质在碳循环中**每日保持恒定**（约1.5g/kg），主要变动碳水与脂肪的比例。

## 七、计算流程总结

```
输入参数：
  ├── 性别 (男/女)
  ├── 年龄 (岁)
  ├── 身高 (cm)
  ├── 体重 (kg)
  ├── 体脂率 (%) [可选]
  └── 活动系数 (1.20~1.90)

↓↓↓

Step 1: 计算 BMR
  BMR = Mifflin-St Jeor公式

Step 2: 计算 TDEE
  TDEE = BMR × 活动系数

Step 3: 设定减重目标
  期望月减重(kg) → 月热量缺口 = 月减重 × 7700
  → 日热量缺口 = 月热量缺口 / 30
  → 日摄入目标 = TDEE - 日热量缺口
  → 校验：不低于1200/1500 kcal（女/男）

Step 4: 选择饮食模式 → 计算三大营养素
  ├── 低碳: 碳水1.5-2g/kg, 蛋白1.5-1.8g/kg, 脂肪补足
  ├── 高碳: 碳水2.5-3.5g/kg, 蛋白1.1-1.5g/kg, 脂肪少量
  └── 碳循环: 按日类型分配不同碳水/脂肪比例

Step 5: 食物换算
  根据三大营养素目标，匹配食物数据库中食物的重量
  → 按早餐30%/午餐40%/晚餐30%分配
  → 输出每餐推荐食物及重量
```

## 六、快速减重 vs 规范减脂：双轨策略

### 6.1 核心理念区别

| 维度 | 快速减重（Rapid Weight Loss） | 规范减脂（Body Recomposition） |
|------|------------------------------|-------------------------------|
| **目标** | 短时间内最大化体重秤数字下降 | 减少脂肪、维持/增加瘦体重（肌肉） |
| **关注指标** | 体重（kg） | 体脂率、腰围、腰臀比、去脂体重 |
| **本质** | 制造巨大热量缺口 | 精准营养分配 + 抗阻训练 |
| **见效时间** | 快（1-2周可见体重下降） | 慢（2-4周开始见围度变化） |
| **主要减掉** | 水分+糖原+部分脂肪+部分肌肉 | 主要为脂肪 |
| **反弹风险** | 高（约71%在3年内反弹） | 低（基础代谢得以维持） |

### 6.2 快速减重：热量缺口策略

- **日缺口范围**：500-1000 kcal/天
- **特点**：初期体重下降快，但约25%来自水分和肌肉
- **适用场景**：短期冲刺（婚礼、拍照等），不适合长期执行
- **安全底线**：女性 ≥1200 kcal/天，男性 ≥1500 kcal/天

#### 储水食物（需严格限制）

人体每储存1克糖原，同时锁住约3-4克水分；1克盐锁水约100ml。以下食物易导致水肿：

| 类别 | 食物举例 | 原因 |
|------|---------|------|
| 🚫 高钠加工食品 | 腊肉、香肠、咸菜、火锅底料、方便面 | 钠含量极高→水钠潴留 |
| 🚫 精制碳水 | 白米饭、白面条、馒头、包子、糕点 | 快速升糖→胰岛素↑→肾脏储钠→水肿 |
| 🚫 高糖食品 | 糖果、含糖饮料、饼干、蛋糕 | 胰岛素飙升→钠重吸收↑ |
| 🚫 重口味酱料 | 酱油、蚝油、豆瓣酱、烧烤酱 | 隐形高钠来源 |

#### 排水消肿食物（推荐多摄入）

| 类别 | 食物举例 | 作用机制 |
|------|---------|---------|
| 🌊 高钾蔬菜 | 冬瓜、菠菜、芹菜、黄瓜、番茄 | 钾促进钠排出，平衡电解质 |
| 🌊 菌菇类 | 口蘑（钾含量≈香蕉12倍）、香菇 | 高钾低钠，强力排水 |
| 🌊 海藻类 | 海带、紫菜 | 富含钾和碘，促代谢 |
| 🌊 杂粮豆类 | 薏米、红豆、绿豆 | 利尿消肿，传统食疗 |
| 🌊 高钾水果 | 香蕉、牛油果、奇异果 | 补钾排钠 |

### 6.3 规范减脂：热量缺口上限（500 kcal/天）

#### 科学依据

华南师范大学体育科学学院荟萃分析指出：**平均每天500千卡的能量亏空，既不会导致瘦体重减少，也不会导致瘦体重增加**——500 kcal 是保留肌肉的"临界点"。

北京体育大学2024年《Sports》期刊随机对照试验证实：500 kcal热量缺口**必须配合运动**才能保留肌肉：
- 仅热量限制组：肌肉量显著下降 -1.21 kg
- 热量限制+运动组：肌肉量与对照组无显著差异

**结论**：规范减脂每日缺口 **不超过500 kcal**，配合 ≥1.6 g/kg 蛋白质和每周 ≥90分钟中高强度运动。

#### 蛋白质需求量上调

| 人群 | 推荐量 | 说明 |
|------|--------|------|
| 快速减重期 | 1.2-1.5 g/kg/天 | 基础防肌肉流失 |
| **规范减脂期** | **1.6-2.2 g/kg/天** | 最大化保留/增加肌肉 |
| 增肌/运动员 | 1.6-2.5 g/kg/天 | 专业训练配合 |

规范减脂期肉类推荐比例上调：每日畜禽肉+鱼虾合计 **150-200g**，蛋类1-2个，配合大豆制品。

### 6.4 腰围与腰臀比（WHR）—— 中国标准

依据国家卫健委《肥胖症诊疗指南（2024年版）》及中国肥胖问题工作组共识：

#### 腰围标准（肋骨下缘与髂嵴连线中点水平围长）

| 分类 | 男性 | 女性 |
|------|------|------|
| **正常** | < 85 cm | < 80 cm |
| 中心性肥胖前期 | 85-90 cm | 80-85 cm |
| **中心性肥胖** | ≥ 90 cm | ≥ 85 cm |

#### 腰臀比（WHR = 腰围 ÷ 臀围）

| 性别 | 正常 | 中心性肥胖风险 |
|------|------|---------------|
| 男性 | < 0.90 | ≥ 0.90 |
| 女性 | < 0.85 | ≥ 0.85 |

#### 其他辅助指标

- **腰围身高比**：腰围(cm) ÷ 身高(cm) ≥ 0.5 → 内脏脂肪超标（男女通用）
- **体脂率**：男性 ≥25%、女性 ≥30% 为体脂过多

> **实践建议**：规范减脂期间，每周测量腰围和臀围各一次（晨起空腹、排便后），体重每周称1-2次即可。腰围缩小比体重下降更能反映真正的减脂效果。

### 6.5 双轨策略对比总结

| 维度 | 快速减重 | 规范减脂 |
|------|---------|---------|
| 日热量缺口 | 500-1000 kcal | ≤500 kcal |
| 蛋白质 | 1.2-1.5 g/kg | 1.6-2.2 g/kg |
| 碳水策略 | 低碳为主，避免精制碳水 | 中等碳水，训练日可高碳 |
| 脂肪 | 低脂（<25%） | 中等（25-30%），优质脂肪 |
| 水分管理 | 严格控钠+高钾食物排水 | 正常饮水，不需刻意排水 |
| 运动推荐 | 有氧为主（加速消耗） | 力量+有氧结合（保留肌肉） |
| 衡量标准 | 体重（每日/隔日称） | 腰围+臀围（每周）+体重参考 |
| 持续周期 | 2-6周短期冲刺 | 3-12个月长期规划 |
| 反弹风险 | 高 | 低 |
